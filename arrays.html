PHP — Arrays (Temas Generales)

En PHP, un array es una estructura que almacena múltiples valores en una sola variable. Internamente es un mapa ordenado: las claves pueden ser enteras (indexado) o strings (asociativo).

Este documento es el marco general. La idea es usarlo como base y luego crear páginas/temas específicos (p. ej., “Arrays asociativos”, “Funciones de arrays”, “Arrays multidimensionales”, etc.).

Tipos de arrays

TipoDescripciónEjemplo (clave ⇒ valor)
Indexado Claves enteras automáticas (0..n) [0 ⇒ "manzana", 1 ⇒ "banana", 2 ⇒ "naranja"]
Asociativo Claves tipo string ["nombre" ⇒ "Pablo", "edad" ⇒ 30]
Multidimensional Arrays dentro de arrays [ ["nombre" ⇒ "Ana"], ["nombre" ⇒ "Luis"] ]

Sintaxis básica

Creación

<?php
// Indexado
$frutas = ["manzana", "banana", "naranja"];

// Asociativo
$persona = [
  "nombre" => "Pablo",
  "edad"   => 30,
  "rol"    => "docente"
];
?>

Acceso por clave/índice

<?php
echo $frutas[0];        // manzana
echo $persona["edad"];  // 30
?>

Operaciones comunes

<?php
$colores = ["rojo", "verde"];

// Agregar al final
$colores[] = "azul"; // ["rojo","verde","azul"]

// Reasignar por índice
$colores[1] = "amarillo"; // ["rojo","amarillo","azul"]

// Eliminar (deja "hueco" si quitás por índice)
unset($colores[0]); // [1 => "amarillo", 2 => "azul"]
?>

Nota Después de unset(), las claves no se reindexan automáticamente. Podés usar array_values() si necesitás reindexar.

Recorrido e iteración

foreach (solo valores)

<?php
$frutas = ["manzana","banana","naranja"];
foreach ($frutas as $f) {
  echo $f . "\n";
}
?>

foreach (clave y valor)

<?php
$persona = ["nombre" => "Pablo", "edad" => 30];
foreach ($persona as $clave => $valor) {
  echo $clave . ": " . $valor . "\n";
}
?>

Arrays multidimensionales

<?php
$alumnos = [
  ["nombre" => "Ana",  "nota" => 9],
  ["nombre" => "Luis", "nota" => 8],
];

echo $alumnos[0]["nombre"]; // Ana

// Recorrer
foreach ($alumnos as $alumno) {
  echo $alumno["nombre"] . " - " . $alumno["nota"] . "\n";
}
?>

Buenas prácticas

  • Elegí nombres de claves claros (en asociativos).
  • Preferí foreach para recorrer; es más legible.
  • Reindexá si necesitás índices consecutivos tras unset() usando array_values().
  • Para colecciones complejas, considerá clases/objetos en lugar de arrays anidados muy profundos.
  • Usá funciones de arrays (evitá reinventar la rueda).

Errores comunes

  • Acceder a una clave/índice que no existe (avisar con isset() o array_key_exists()).
  • Confundir sort() (reindexa) con asort() (preserva claves).
  • Olvidar que unset() no reindexa.
  • Mezclar sin control tipos de claves (enteras/strings) en el mismo array.

Ejercicios sugeridos

  1. Crear un array indexado de 5 colores y ordenarlo alfabéticamente.
  2. Crear un array asociativo de “alumno” con nombre, edad y promedio; imprimir “clave: valor”.
  3. Armar un array multidimensional de productos (nombre, precio) y calcular el total con array_reduce().
  4. Filtrar de un array de números solo los pares usando array_filter().
