Variables en PHP

Una variable en PHP es un contenedor que almacena datos para usarlos en el programa. Todas las variables comienzan con el símbolo $ y pueden contener texto, números, booleanos, arrays, objetos o incluso NULL.

Reglas generales

  • Deben comenzar con $.
  • El nombre empieza con una letra o guion bajo, seguido de letras/números.
  • No pueden contener espacios ni caracteres especiales.
  • Son sensibles a mayúsculas/minúsculas: $nombre ≠ $Nombre.

Ejemplo básico

<?php
$nombre = "Pablo";
$edad = 30;
$activo = true;

echo "Mi nombre es $nombre\n";
echo "Tengo $edad años\n";
echo "¿Estoy activo? " . ($activo ? "Sí" : "No");
?>

Actualizar valores

<?php
$contador = 1;
$contador = $contador + 1; // ahora vale 2
$contador += 5;            // ahora vale 7
echo $contador;
?>

Concatenación de strings

<?php
$nombre = "Pablo";
$mensaje = "Hola, " . $nombre . "!";
echo $mensaje; // Hola, Pablo!
?>

Buenas prácticas

  • Usar nombres descriptivos ($usuario, $precioTotal).
  • Inicializar siempre las variables antes de usarlas.
  • Usar var_dump($var) o print_r($var) para depuración.
  • Escapar las variables al imprimir en HTML con htmlspecialchars().
Resumen: las variables en PHP se crean con $, no tienen tipo fijo, y se pueden usar para almacenar cualquier tipo de dato en tu aplicación web.
